Content of the Open Source Version

The main control libraries of UI5 like 'sap.ui.commons', 'sap.ui.ux3', the responsive library 'sap.m' and the entire UI5 Core are now all available under the Apache license and part of the above downloads. These libraries are identical to the non-Open-Source versions. However, some others, like the 'sap.viz' chart library, for example, are not (yet) available under an Open Source license and not contained here.

Sources

To get the source code, just look inside the UI5 Runtime archive, it contains all source files also in non-minified format (*-dbg.js).

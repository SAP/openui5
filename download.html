The main control libraries like 'sap.ui.commons', 'sap.ui.ux3', responsive library 'sap.m' and the entire UI5 Core are now all available under the Apache license. These are identical to the non-Open-Source versions. However, others, like the 'sap.viz' chart library, for example, are not yet available under an Open Source license.

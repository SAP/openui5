Get Ready

All you need to develop Web apps with OpenUI5 is a development environment and a browser. Choose between SAP Web IDE in the cloud for maximum convenience or set up your local development environment for maximum flexibility.

Both options are described under Development Environment in our documentation that is included in the OpenUI5 Demo Kit.

As our project grows, we'll follow the Model View Controller (MVC) approach. In this example, our project now contains three files:

  • index.html
    The entry point for our app bootstraps OpenUI5 as above and places the view into the HTML body.

  • App.view.xml
    The view is written in XML (could also be JavaScript or JSON) and contains presentation logic to display a page and a button control.

  • App.controller.js
    The controller defines an event handler for responding to user actions by displaying a message toast.

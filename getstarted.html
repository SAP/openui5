Get the Basics!

You can use SAPUI5 in any HTML file with the following bootstrap script tag (make sure that you have put the UI5 runtime download in your application folder, if you reference it like in this example): 

<script id="sap-ui-bootstrap"
   src="resources/sap-ui-core.js"
   data-sap-ui-theme="sap_bluecrystal"
   data-sap-ui-libs="sap.ui.commons"></script>

A simple application looks like this: 

<!DOCTYPE html>
<html><head>
    <meta http-equiv='X-UA-Compatible' content='IE=edge' />
    <title>Hello World</title>

    <script id='sap-ui-bootstrap'
        src='resources/sap-ui-core.js'
        data-sap-ui-theme='sap_bluecrystal'
        data-sap-ui-libs='sap.ui.commons'></script>

<script>
    var btn = new sap.ui.commons.Button({
        text:'Press me!',
        press: function() {
            alert("Hello!")
        }
    });
    btn.placeAt('content');
</script>

</head>
<body class='sapUiBody'>
    <div id='content'></div>
</body>
</html>

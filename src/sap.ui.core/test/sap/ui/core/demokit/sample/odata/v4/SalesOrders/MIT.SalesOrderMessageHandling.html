Manual Integration Test: Message Handling

DESCRIPTION

Test message handling with the Sales Orders application.

PREPARATION

To check console errors open the developer tools before starting the application and check with each step, that there are no errors and warnings.

Testing with mock data

Start the Sales Orders application with following URL (adjust host and port if necessary):
  • local: http://localhost:8080/testsuite/test-resources/sap/ui/core/demokit/sample/common/index.html?component=odata.v4.SalesOrders

REALIZE and CHECK

Bound messages in Sales Orders application

  1. Check: On the Show messages button the number of messages is 2.
  2. Check: The Note field of the sales order with the ID 0500000001 has a colored border indicating a warning.
  3. Check: When clicking into the Note field of the sales order with the ID 0500000001 the warning "Enter customer reference if available" is displayed.
  4. Click on the Show Messages button.
  5. Check: A popover is diplayed containing 2 messages, an information message with the text "Example for an unbound message" and a warning message with the text "Enter customer reference if available".
  6. Click on the message "Example for an unbound message" to get to the message details.
  7. Check: The content of the message details is the same as you get with the URL http://raw.githubusercontent.com/SAP/openui5/master/NOTICE.txt.
  8. Go back to the messages overview.
  9. Click on the message "Enter customer reference if available" to get to the message details.
  10. Check: The content of the message details is: Details for "Enter customer reference if available" (relative longtext URL).
  11. Go back to the messages overview.
  12. Close the message popover.
  13. Check: On the Show messages button the number of messages is now 0.
  14. Check: The Note field of the sales order with the ID 0500000001 does not have a colored border any more.
  15. Click on the Show Messages button.
  16. Check: A popover is diplayed containing 0 messages.
  17. Close the message popover.
  18. In the New Sales Orders table select the sales order with the ID 0500000001.
  19. Check: On the Show messages button the number of messages is 2.
  20. Check: The Note field of the sales order with the ID 0500000001 has a colored border indicating a warning.
  21. Check: When clicking into the Note field of the sales order with the ID 0500000001 the warning "Enter customer reference if available" is displayed.
  22. Check: The Note field in the Sales Order Details section has a colored border indicating a warning.
  23. Check: When clicking into the Note field in the Sales Order Details section the warning "Enter customer reference if available" is displayed.
  24. Check: The Quantity field of the sales order line item with the item position 0000000020 has a colored border indicating an error.
  25. Check: When clicking into the Quantity field of the sales order line item with the item position 0000000020 the error "Minimum order quantity is 2" is displayed.
  26. Click on the Show Messages button.
  27. Check: A popover is diplayed containing 2 messages, a warning message with the text "Enter customer reference if available" and an error message with the text "Minimum order quantity is 2".
  28. Click on the message "Minimum order quantity is 2" to get to the message details.
  29. Check: The content of the message details is: Details for "Minimum order quantity is 2" (absolute longtext URL).
  30. Go back to the messages overview.
  31. Close the message popover.
  32. Check: On the Show messages button the number of messages is 1.
  33. Click on the Show Messages button.
  34. Check: A popover is diplayed containing 1 message, an error message with the text "Minimum order quantity is 2".
  35. Close the message popover.
  36. Check: The Note field of the sales order with the ID 0500000001 does not have a colored border any more.
  37. Check: The Note field in the Sales Order Details section does not have a colored border any more.
  38. Check: The Quantity field of the sales order line item with the item position 0000000020 still has a colored border indicating an error.
  39. In the New Sales Orders table select the sales order with the ID 0500000000.
  40. Check: There is no input field with a colored border in the Sales Order Line Items table.
  41. Check: On the Show messages button the number of messages is still 1.
  42. In the New Sales Orders table select again the sales order with the ID 0500000001.
  43. Check: The Quantity field of the sales order line item with the item position 0000000020 again has a colored border indicating an error.
  44. Check: On the Show messages button the number of messages is 1.
  45. Click on the Show Messages button.
  46. Check: A popover is diplayed containing 1 message, an error message with the text "Minimum order quantity is 2".
  47. Close the message popover.

Manual Integration Test: List Binding Template w/ Annotation Changes

DESCRIPTION

Test of the List Binding Template application.

PREPARATION

To check console errors open the developer tools before starting the application and check with each step, that there are no errors and warnings.

Testing with mock data

Start the List Binding Template application with following URL (adjust host and port if necessary):
  • as part of the demo kit: http://<host>:<port>/demokit/test-resources/sap/ui/core/demokit/sample/common/index.html?component=odata.v4.ListBindingTemplate
  • local: http://<host>:<port>/test-resources/sap/ui/core/demokit/sample/common/index.html?component=odata.v4.ListBindingTemplate

Testing with back end

Configure your IDE to be able to run the List Binding Template application against an ABAP back end system.
Start the List Binding Template application with the URL (adjust host and port if necessary):
  • local: http://<host>:<port>/test-resources/sap/ui/core/demokit/sample/common/index.html?component=odata.v4.ListBindingTemplate&realOData=true

REALIZE and CHECK

  • Choose whether to use mock data or a real back end. Start the application once as described above and then once more in a new window with the additional URL parameter "&setAnnotationChangePromise".

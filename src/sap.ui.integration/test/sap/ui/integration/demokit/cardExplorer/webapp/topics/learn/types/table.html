Table Card

The table card is used to display multiple items in table view.

Properties

The table card contains an array of rows.

Table row properties

Property Type Required Description Schema Version
columns array No Defines the columns attributes. 1.15.0
actions object No Defines actions that can be applied on the item. 1.15.0

Table column properties

Property Type Required Description Schema Version
title string No Defines language-dependent title of the column. 1.15.0
width string No Defines the width of the column. 1.15.0
value string No Represents the text value of the column. 1.15.0
icon icon No Represents column with icon. 1.15.0
state string No Defines the state of the column. 1.15.0
url string No Defines the URL string. 1.15.0
target string No Specifies where to open the 'url', if it is provided. 1.15.0
identifier string No Defines whether the column is an identifier. 1.15.0
progressIndicator string No Defines the progress indicator attributes. 1.15.0

Example

Define the type and data for the card: 

{
	"sap.app": {
		"id": "card.explorer.table.card",
		"type": "card"
	},
	"sap.card": {
		"type": "Table",
		"header": {
			"title": "Sales Orders for Key Accounts",
			"subTitle": "Today",
			"status": {
				"text": "5 of 27"
			}
		},
		"content": {
			"data": {
				"request": {
					"url": "./cardcontent/tableitems.json"
				}
			},
			"row": {
				"columns": [
					{
						"label": "Sales Order",
						"value": "{salesOrder}",
						"identifier": true
					},
					{
						"label": "Customer",
						"value": "{name}"
					},
					{
						"label": "Net Amount",
						"value": "{netAmount}"
					},
					{
						"label": "Status",
						"value": "{status}",
						"state": "{statusState}"
					},
					{
						"label": "Avatar",
						"width": "12%",
						"icon": {
							"src": "{icon}"
						}
					},
					{
						"label": "Progress",
						"progressIndicator": {
							"percent": "{percent}",
							"text": "{percentValue}",
							"state": "{progressState}"
						}
					}
				]
			}
		}
	}
}

	<w:Card manifest="./manifest.json" width="400px" height="auto"/>

Integration of Cards

The integration of cards with the host environment requires additional logic such as resolving card destinations, handling card actions, persisting configuration changes, etc. For more information, read the Integrate section.

Besides the card manifest, the samples in this section illustrate and include specific host integration logic. Therefore, they are not download-and-reuse ready and first should be adapted to your specific host environment. If you want to use them in the SAP Business Application Studio, they have to be imported there as an application.

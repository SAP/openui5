Actions Strip

Actions strip contains buttons or links in an overflow toolbar. Visibility and order can be set according to their priority. Currently available in List card and card footer.

Experimental feature. The API may change.

Usage

Actions strip items can be static or dynamic based on the data.

actionsStrip ActionsStrip|ActionsStripItem[] Yes Binding info object or array of static items. 1.61.0 1.121

ActionsStripItem Properties
Property Type Default Value Required Description Schema Version Since

  • "Button" - creates a button
  • "ToolbarSpacer" - creates a toolbar spacer
  • "Link" - creates a link with optional icon. Available since 1.107
  • "Label" - creates a label for one item or for a group of items. Available since 1.121 Note: This property is experimental and may change!
 1.35.0 1.93

visible boolean true No Determines whether the item is visible. 1.35.0 1.93

Parent Model

When actions strip items are created via item template, you can bind them to the parent context using the model called "parent", like: "{parent>/parentProperty}"

Examples

Actions strip with static items:

{
	"sap.card": {
		"type": "List",
		"data": {
			...
		},
		"header": {
			"title": "Title"
		},
		"content": {
			"item": {
				"title": "{name}",
				"description": "{description}",
				"actionsStrip": [
					{
						"type": "Link",
						"text": "Agenda",
						"icon": "sap-icon://action",
						"actions": [
							{
								"type": "Navigation",
								"parameters": {
									"url": "{agendaUrl}"
								}
							}
						]
					},
					{
						"type": "ToolbarSpacer"
					},
					{
						"text": "Approve",
						"overflowPriority": "High",
						"actions": [
							{
								"type": "Custom",
								"parameters": {
									"method": "approve"
								}
							}
						]
					}
				]
			}
		}
	}
}

Actions strip with dynamic items:

{
	"sap.card": {
		"type": "List",
		"data": {
			"json": {
				"items": [
					{
						"title": "Item 1",
						"actionItems": [
							{
								"text": "Accept"
							},
							{
								"text": "Reject"
							}
						]
					},
					{
						"title": "Item 1",
						"actionItems": [
							{
								"text": "Accept"
							},
							{
								"text": "Reject"
							},
							{
								"text": "Remind Me Later"
							}
						]
					}
				]
			}
		},
		"header": {
			"title": "Title"
		},
		"content": {
			"data": {
				"path": "items"
			},
			"item": {
				"title": "{name}",
				"description": "{description}",
				"actionsStrip": {
					"item": {
						"template": {
							"text": "{text}",
							"id": "{parent>/id}"
						},
						"path": "actionItems"
					}
				}
			}
		}
	}
}

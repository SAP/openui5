Loading Placeholders

Loading placeholders inform the user that a card is performing a background task, such as fetching data from the backend.

Loading Placeholder Properties

Property Type Required Description Schema Version Since
delay Integer No The delay in milliseconds, after which the loading placeholders will show up. This delay applies only for the initial loading. 1.56.0 1.134

Example

An example of a card with delay of the loading placeholder:

{
	"sap.card": {
		"type": "List",
		"configuration" : {
			"loadingPlaceholders": {
				"delay": 2000
			}
		},
		"header": {
			"title": "Some title",
			"subtitle": "Date: {parameters>/TODAY_ISO}"
		},
		...
	}
}

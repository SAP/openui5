Integration Card Headers

The card header is used to group a set of card attributes.

This section describes the different header types that can be used in the card header area ("sap.card"/header in the manifest).

Note: "sap.card"/header is mandatory. Providing it together with title is essential for proper screen reader support and other functionalities. If needed, it can be visually hidden using its visible property.

Header Types

There are two types of card headers:

  • Default
  • Numeric

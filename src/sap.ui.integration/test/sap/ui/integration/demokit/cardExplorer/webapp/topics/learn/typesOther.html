Integration Card Types

Other Card Types

This section describes the different content types that can be used in the card content area ("sap.card"/content in the manifest).

The card types listed below offer more flexibility than declarative cards, but might increase efforts on development side.
Those cards are not considered to support a native mobile (iOS/Android) experience in the future.

All below cards are delivered with the OpenUI5 distribution.

  • Adaptive
  • Component
  • WebPage

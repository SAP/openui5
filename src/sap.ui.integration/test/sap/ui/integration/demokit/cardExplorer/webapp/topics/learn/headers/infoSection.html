Info Section

Experimental feature since version 1.136. The API may change.

Info Section is a flexible block inside the header which can hold various other components, like a time stamp, rating, statuses, etc.
Note: Currently only items of type "Status" are available.

Example

"header": {
	"title": "Project Cloud Transformation",
	"subTitle": "Revenue",
	"unitOfMeasurement": "EUR",
	"infoSection": {
	"rows": [
	  {
		"items": [
		  {
			"type": "Status",
			"value": "On Track",
			"state": "Success",
			"inverted": true,
			"showStateIcon": true
		  },
		  {
			"type": "Status",
			"value": "OKR Relevant",
			"state": "None",
			"inverted": true,
			"showStateIcon": true
		  },
		  {
			"type": "Status",
			"value": "Goal Template Available",
			"state": "Information",
			"inverted": true,
			"showStateIcon": true
		  }
		]
	  }
	]
  }
}

Testing Overview

This is an overview page of various ways to test the generated app during development.
Choose one of the access points below to launch the app as a standalone application.

  • index.html - start the app (with real service)
  • test/unit/unitTests.qunit.html - run all unit tests
  • test/integration/opaTestsComponent.qunit.html - run integration tests (component loader)
  • test/integration/opaTestsIFrame.qunit.html - run integration tests (iframe loader)
  • test/integration/opaTestsGherkinComponent.qunit.html - run integration tests (component loader)
  • test/integration/opaTestsGherkinIFrame.qunit.html - run integration tests (IFrame loader)
  • run this in phone emulation mode! test/integration/opaTestsPhone.qunit.html - run all integration tests for phone

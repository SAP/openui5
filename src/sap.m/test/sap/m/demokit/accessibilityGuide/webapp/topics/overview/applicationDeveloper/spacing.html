Spacing

In accordance with the WCAG 2.2 Target Size Criterion 2.5.8, application developers have to ensure that targets meet a minimum size or have sufficient spacing around them. This helps avoid cases where users try to activate an interactive element but accidentally activate another element.

  • The size of the target for pointer inputs is at least 24 by 24 CSS pixels.
  • Position any undersized targets (those less than 24 by 24 CSS pixels) carefully. Ensure that if you center a 24 CSS pixel diameter circle on the bounding box of each target, the circles don't intersect with another target or with the circle of another undersized target.

Controls Displayed as Links

Those controls can display links as part of sentences or paragraphs. In such cases, the target size of the link is constrained by the line height of the non-target text:

  • sap.m.Link
  • sap.m.ObjectStatus
  • sap.m.ObjectNumber
  • sap.m.ObjectIdentifier
  • sap.m.ObjectMarker

In all other cases you need to ensure that each interactive element has no other interactive elements nearby.

For example, in the following snippet we've increased the horizontal spacing between the links in the context of a sap.m.OverflowToolbar by providing additional left and right margins with the help of the sapUiTinyMarginBeginEnd class:

<OverflowToolbar>
	<ObjectIdentifier
		class="sapUiTinyMarginBeginEnd"
		titleActive="true"
		title="HT-2001"
	/>
	<Link
		class="sapUiTinyMarginBeginEnd"
		text="10 Portable DVD player"
	/>
	<ObjectStatus
		class="sapUiTinyMarginBeginEnd"
		active="true"
		state="Success"
		text="Product Shipped"
		icon="sap-icon://sys-enter-2"
	/>
	<ObjectNumber
		class="sapUiTinyMarginBeginEnd"
		active="true"
		number="150"
		state="Success"
		unit="EUR"
	/>
</OverflowToolbar>

Network Graph

For Network Graph, to address the WCAG 2.2 Target Size Criterion 2.5.8, we recommend setting a minimum zoom level of 125%.

XML View

<Graph id="graph">

Controller

this.byId("graph").attachGraphReady(function() {
	this.zoom({
		zoomLevel: 1.25
	});
});

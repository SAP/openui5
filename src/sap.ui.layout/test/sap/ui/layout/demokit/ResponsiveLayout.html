ResponsiveLayout

Introduction

The ResponsiveLayout can be attached to the Form Control in order to arrange the Elements in a responsive way (see ResponsiveFlowLayout). The size and arrangement of the single elements depends on the size of the screen. The Form Control consists of a Form on top level which contains several FormContainers with several FormElements in it. According to our layout concept all of these can have LayoutData attached to them.

You can use ResponsiveFlowLayoutData on FormContainers, FormElements and the content controls like in the ResponsiveFlowLayout control. So it is possible to influence the linebreaks, min width and weights of the single elements.If you use ResponsiveFlowLayoutData on Containers or Elements you should disable the margins to ayoid nested margins. if ResponsiveFlowLayoutData are used, the margins are disabled by default.
